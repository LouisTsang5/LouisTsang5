## 🛠️ Featured Projects

### [Rust Gameboy Emulator](https://github.com/LouisTsang5/rust-gb-emu)
A fun personal project implementing a full **Game Boy hardware emulator** in Rust.  
- Accurate 16-bit 8080-like CPU emulation  
- Focus on performance and correctness

### [Clipboard Server](https://github.com/LouisTsang5/clipboard-server)
A secure clipboard sharing tool consisting of a Rust server and client.  
- TCP/IP socket communication  
- AES-GCM encryption + zlib compression for secure data transfer

### [Rust HTTP Server](https://github.com/LouisTsang5/rust-hmp-server)
A lightweight, high-performance HTTP server written in Rust for API mocking and static file serving.  
- Custom HTTP parser  
- Multi-threaded async I/O  
- File caching with kernel event-based invalidation

### [Rust Port Forward](https://github.com/LouisTsang5/rust-portforward)
A modern, efficient TCP port forwarding tool written in Rust.  
- Multi-threaded async I/O  
- Supports forwarding multiple TCP ports simultaneously  
*(Rust rewrite of the C version below)*

### [TCP Port Forward (C)](https://github.com/LouisTsang5/c-socket/blob/main/tcp-portforward.c)
Multi-threaded TCP port forwarding utility written in C.

### [Boot ASM](https://github.com/LouisTsang5/boot-asm)
A personal low-level project implementing a **BIOS boot loader** in x86 assembly.

### [Transaction Recorder](https://github.com/LouisTsang5/trans-rec)
A clean, client-side personal finance tool built with TypeScript.  
- Full frontend application (no backend required)  
- JSON-based data export/import  
- Monthly spending reports and visualizations
